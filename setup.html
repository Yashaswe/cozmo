Materials Needed

  • 1 Anki Cozmo robot
  • 1 Iphone (we tried to use an Android phone but it did not work)
  • At least one computer with the below software installed on it with a way of connecting to an Android phone
  • A cable to connect the phone to the computer, probably micro USB or USB-C

Software Prerequisites

  • The Cozmo app
  • Python 3.10 or higher
  • Visual Studio Code
  • Python libraries to import:

Setup Instructions

  • For each computer you would like to be able to actually run programs on the robot from, you must install the android debug bridge and the cozmo SDK
  • If Python is not installed already, go to https://www.python.org/downloads/ and download the latest version available, or at least 3.10.
  • Download and install VSCode from the link https://code.visualstudio.com/download. We preferred to use VSCode for our project since it has particularly good Jupyter Notebook integrations.
